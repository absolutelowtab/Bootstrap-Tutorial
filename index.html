Что такое Bootstrap?

Bootstrap — это самый популярный CSS-фреймворк для создания адаптивных и мобильных веб-сайтов. С его помощью можно быстро создавать современные интерфейсы без глубоких знаний CSS.

В этом учебнике мы рассмотрим основные концепции:

  • Система сеток — 12-колоночная адаптивная сетка
  • Компоненты — готовые UI элементы
  • Утилиты — вспомогательные классы
  • Адаптивный дизайн — mobile-first подход
  • Кастомизация — настройка через Sass переменные

Bootstrap стал стандартом в веб-разработке. Он позволяет создавать профессиональные интерфейсы в разы быстрее, обеспечивая кроссбраузерность и адаптивность.

<!-- Пример сетки Bootstrap -->

  <div class="row">
    <div class="col-md-4">
      <div class="card">
        <div class="card-body">
          <h5 class="card-title">Заголовок карточки</h5>
          <p class="card-text">Текст карточки</p>
          <a href="#" class="btn btn-primary">Кнопка</a>
        </div>
      </div>
    </div>
  </div>
</div>

Система сеток Bootstrap

12-колоночная адаптивная сетка — основа Bootstrap.

Классы сетки

Bootstrap использует систему из 12 колонок с шестью контрольными точками:

.col-

Колонки для экранов менее 576px (xs). Например: .col-6 создаёт колонку шириной 6/12 (50%).

.col-sm-

Колонки для экранов ≥576px (small). Применяется от 576px и выше.

.col-md-

Колонки для экранов ≥768px (medium). Используется для планшетов.

.col-lg-

Колонки для экранов ≥992px (large). Для ноутбуков и десктопов.

Mobile-first подход

Bootstrap использует mobile-first подход. Стили сначала пишутся для мобильных, затем добавляются медиа-запросы для более крупных экранов.

<div class="col-12 col-md-6 col-lg-4">
  
</div>

Структура сетки

Сетка Bootstrap состоит из трёх основных элементов:

Контейнер

  • .container — центрированный контейнер с фиксированной шириной
  • .container-fluid — контейнер на всю ширину экрана
  • .container-{breakpoint} — адаптивный контейнер

Строки

  • .row — горизонтальная группа колонок
  • .row-cols-* — автоматическое количество колонок
  • .g-* — отступы (gaps) между колонками

Колонки

  • .col — автоматическая ширина
  • .col-* — явная ширина (1-12)
  • .col-auto — ширина по содержимому

Важно: Все колонки должны быть внутри .row, а все строки внутри .container или .container-fluid.

Компоненты Bootstrap

Готовые UI элементы для быстрой разработки интерфейсов.

Основные компоненты

Bootstrap включает десятки готовых компонентов. Вот самые популярные:

Кнопки

Стилизованные кнопки с вариантами цвета, размера и состояния.

<button class="btn btn-primary">
  Основная кнопка
</button>

Карточки

Гибкие контейнеры для контента с заголовками, изображениями и кнопками.

<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">Заголовок</h5>
    <p class="card-text">Текст карточки</p>
  </div>
</div>

Оповещения

Контекстные сообщения обратной связи для пользователей.

<div class="alert alert-success">
  Операция выполнена успешно!
</div>

Навигация

Адаптивные навигационные панели, табы и пагинация.

<nav class="navbar navbar-expand-lg">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Brand</a>
  </div>
</nav>

Интерактивные компоненты

Bootstrap также включает интерактивные компоненты с JavaScript:

Модальные окна

Всплывающие окна для диалогов, форм и уведомлений.

Выпадающие списки

Контекстные меню для навигации и действий.

Сворачиваемые элементы

Показать/скрыть контент с анимацией.

Карусели

Слайд-шоу для циклического перебора элементов.

JavaScript инициализация

Для работы интерактивных компонентов нужно подключить JavaScript Bootstrap и инициализировать их.

// Инициализация модального окна
var myModal = new bootstrap.Modal(document.getElementById('myModal'));
myModal.show();

Утилиты Bootstrap

Вспомогательные классы для быстрой стилизации.

Классы утилит

Утилиты Bootstrap позволяют стилизовать элементы без написания кастомного CSS:

Синтаксис утилит:

/* Отступы */
.m-*  /* margin */
.p-*  /* padding */
.mt-* /* margin-top */
.pb-* /* padding-bottom */

/* Текст */
.text-*      /* цвет текста */
.fw-*        /* жирность */
.fs-*        /* размер шрифта */
.text-*      /* выравнивание */

/* Фон */
.bg-*        /* цвет фона */

/* Flexbox */
.d-flex      /* display flex */
.justify-*   /* выравнивание по главной оси */
.align-*     /* выравнивание по поперечной оси */

/* Границы */
.border      /* граница */
.rounded     /* скругление */
.shadow      /* тень */

Размеры и отступы

Bootstrap использует систему размеров от 0 до 5 для отступов:

0

0rem (0px) — отсутствие отступа.

1

0.25rem (4px) — маленький отступ.

2

0.5rem (8px) — стандартный отступ.

3

1rem (16px) — средний отступ.

4

1.5rem (24px) — большой отступ.

5

3rem (48px) — очень большой отступ.

Автоматизация с помощью утилит

Все утилиты можно комбинировать для создания сложных стилей без CSS:

<div class="d-flex justify-content-between 
            align-items-center 
            p-3 
            bg-light 
            border 
            rounded 
            shadow-sm">
  <span class="fw-bold text-primary">Заголовок</span>
  <button class="btn btn-success btn-sm">Действие</button>
</div>

Демонстрация утилит:

Макеты и адаптивность

Создание адаптивных макетов с помощью Bootstrap.

Breakpoints (Контрольные точки)

Bootstrap использует шесть контрольных точек для адаптивного дизайна:

XS

<576px

Мобильные телефоны (portrait).

SM

≥576px

Мобильные телефоны (landscape).

MD

≥768px

Планшеты.

LG

≥992px

Ноутбуки.

XL

≥1200px

Десктопы.

XXL

≥1400px

Большие мониторы.

Адаптивные классы

Большинство классов Bootstrap имеют responsive варианты:

.col-{breakpoint}-*

Адаптивные колонки сетки.

.d-{breakpoint}-*

Адаптивное отображение (display).

.text-{breakpoint}-*

Адаптивное выравнивание текста.

.m{side}-{breakpoint}-*

Адаптивные отступы.
